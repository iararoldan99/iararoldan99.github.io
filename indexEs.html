Un poco sobre mí y lo que hago 🌈

Me llamo Iara Roldán, tengo 23 años y soy Software Developer, trabajo todos los días con Java y ReactJS.
Soy entusiasta de las buenas prácticas de programación y siempre estoy dispuesta a aprender algo nuevo 💪
El stack de tecnologías que manejo va desde Java (SpringBoot, POO y programación funcional), JavaScript (React, Redux), bases de datos (MySQL, SQL Server) a otras cositas como: UI Libraries (MaterialUI, Bootstrap), CSS - Sass, Node, RabbitMQ, Microservices, Docker, Git, Terminal, Jenkins, Postman, Testing, etc.
Intento mejorar todos los días manteniendome actualizada y aprendiendo de otros programadores y colegas 🤝

Educación 👩🏻‍💻

Empecé a programar en 2020, cuando me inscribí a un Bootcamp de programación para sumar mujeres a sistemas, durante la pandemia. No tenía conocimientos previos en informática, asi que fue bastante desafiante pero me terminó gustando mucho y por eso me quedé acá.
Dos años después, soy estudiante de la tecnicatura en Desarrollo de Software en UADE.
Me anoté a muchos cursos de programación desde que terminé el bootcamp en 2020, como por ejemplo el curso de ReactJS de CoderHouse, o academias virtuales como Codecademy o freecodecamp, entre otros.
Siempre que puedo me pongo a crear proyectos propios con tecnologías que voy aprendiendo.

Experiencia laboral 💻

👩🏻‍💻- Empecé en mi primer trabajo en "Adistec Corp." en octubre de 2020, donde trabajé como Junior Java Developer hasta septiembre de 2022. El stack de tecnologías con los que trabajábamos incluía: Java 8, SpringBoot, MySQL, SQL Server, JavaScript, ReactJS, Redux, MaterialUI, Bootstrap, Sass, etc. Usábamos metodologías ágiles, GIT y JIRA. Aprendí un montón en ese puesto y estoy por siempre agradecida por la increíble experiencia que fue trabajar en Adistec.

👩🏻‍💻- En septiembre de 2022, entré a BBVA Argentina para trabajar como Full Stack Java Developer. Actualmente trabajo ahí.
