A little bit about me and what I do 🌈

My name is Iara Roldan, I'm a 23-year-old Software Developer, working everyday with Java and ReactJS.
I'm very enthusiastic about good programming practices and always willing to learn something new 💪
The stack of technologies I use includes Java (SpringBoot, POO and Functional programming), JavaScript (React, Redux), databases (MySQL, SQL Server) and a little bit of some other stuff such as: UI Libraries (MaterialUI, Bootstrap), CSS - Sass, Node, RabbitMQ, Microservices, Docker, Git, Terminal, Jenkins, Postman, Testing, etc.
